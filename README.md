Organizador de Estudos

Problema
Muitos estudantes têm dificuldade em organizar suas tarefas e manter uma rotina de estudos.

Solução
Esta aplicação permite adicionar e listar tarefas de forma simples pelo terminal.

Público-alvo
Estudantes que precisam organizar seus estudos.

Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Sair do sistema

Tecnologias
- Python
  
Instalação
pip install -r requirements.txt

Como executar
python src/app.py

Testes
python -m pytest

Dependências
pytest
flake8
